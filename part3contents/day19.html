自定义指令：指令是个函数,用来操作dom的, 里面的this 返回window
    a)    Vue.directive('指令名不带v-',函数(el,binding))            全局
        el == 使用指令的DOM元素
        binding 是个对象 含有传入的 参数(binding.value)
    b)  定义在选项里面                                            实例（组件）内部
        directives:{
            指令名不带v-    : 函数(el,binding){}
        }
    执行时机: 指令的钩子函数
        指令：函数(bind,update)|    对象(里面可以有钩子函数)
            bind:fn(el,binding)        指令第一次绑到元素调用,只调用一次
            update:fn    指令所在的位置，数据，模板更新了
            componentUpdated:fn 更新完成
            inserted:fn    被绑定的元素插入到父节点时调用

VUE特点:
    1. 默认情况下  this 指向根实例(组件)  VM
        this.$xxx  全局属性
        this.数据名  返回数据    this.数据=值 修改    操作data里面的数据
            this.$data.数据  获取
        this.方法名()  调用methods里面的方法
            this.$全局方法()
        this.计算属性  返回计算后的值
    2.    Vue 并不能检测到所有的属性被修改（发生场景：根数据对象的属性返回的是基本类型arr[0] 返回的'apple')
        解决：a)标准数据  b)Vue.set( 目标对象, key, value )

    使用：    {{数据名 | 过滤器名(参数1,参数2)}}
            :属性="数据名 | 过滤器名(参数1,参数2)"
    定义:
            a) Vue.filter('过滤器名称',函数(要过滤的数据,参数))    全局
            b) 选项
                filters:{        组件内部的过滤器
                    过滤器名称:函数
                    过滤器名称2:函数(要过滤的数据,参数)
                }

resource：
    Vue.http(url,{配置}).then(成功回调(res),失败回调(res))
    this.$http(url,{配置}).then(成功回调(res),失败回调(res))
    this.$http.get(url,{数据},{配置}).then(成功回调(res),失败回调(res))
    this.$http.post(url,{数据},{配置}).then(成功回调(res),失败回调(res))
    this.$http.jsonp(url,{数据},{配置}).then(成功回调(res),失败回调(res))
    res:
        data:"数据"      string/json/arr
        headers:    请求头
        ok: true/false  数据状态
        request:{ 请求时的配置
            method: "GET",   提交方式
            data: "携带数据",
            params: {携带参数},
            headers: 请求头,
            xhr: ajax对象
            }
        status：200
        statusText： 状态码解释
        emulateJSON:true //允许携带请求头
        jsonp:    "callback"    jsonp的回调key
        timeout: 延时
        crossOrigin: 是否携带跨源凭证

地理定位:    geolocation
移动互联网形态
    社交-微信、微博、陌陌
    团购-美团、饿了么、外卖、滴滴、优步、
    共享-单车，电单车、电汽车
    大数据-分析
定位原理：
    pc:        IP地址(误差,强制)
    phone:    GPS(准确)
js实现地理定位：
    navigator.geolocation    返回 定位对象
    兼容：IE9+     chrome     FF
                FQ    
获取一次地理信息：获取的过程是异步的
    navigator.geolocation.getCurrentPosition(succFn,errorFn)
        errorFn(ev)
            ev.code 错误码
                0    未知错误
                1    用户拒绝
                2    获取失败
                3    超时
            ev.message 错误信息
        succFn(ev)
            坐标     ev.coords
            经度     ev.coords.longitude
            纬度     ev.coords.latitude
            坐标精确度:    ev.coords.accuracy 低于60，基本不可用
            海拔高度    ev.coords.altitude
            海拔高度精确度    ev.coords.altitudeAccuracy
            朝向     ev.coords.heading
            速度    ev.coords.speed
实时获取地理信息：获取的过程是异步的
    timeID = navigator.geolocation.watchPosition(succFn,errorFn,配置)      
        配置:
        {
            timeout:300
            ....
        }
    navigator.geolocation.clearWatch(timerID); 关闭实时获取

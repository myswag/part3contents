混合开发:
    原生app    /    navtive app
        android: java    
        ios: objectC    XCode(IDE)
        优点：性能高，调用用户设备（机行，系统，麦克风，照相机...)
        缺点：需要下载，开发周期长，团队规模大
        场景：大型应用，大型游戏，需要下载
    HTML5+     /     web app
        HTML+CSS+JS
        优点：开发周期短，跨平台，
        缺点：默认无法调取设备信息
        场景：淘宝、京东、网易.....
    hybrid app / 混合app
        html5+ 搭建页面
        通过第三方库 调取设备信息
        优点：开发周期短，跨平台，设备（第三方库）

cordova:
    phoneGap: 软件 第三打包工具、搭环境
        本机虚拟服务器(localhost) - > 手机安装对应APP
                条件： PC 和手机，要在同一个局域网段
                同一个局域网段: 同一个路由
        搭环境:
            pc:    localhost web服务器  装pc端的phoneGap
            手机: 查看效果 phoneGap app
        安装：https://phonegap.com/products/#desktop-app-section  PC
              ios->appStore->搜索->phoneGap->下载
        创建项目:
            + ->    create -> 选择模板 ->     local path (项目路径) -> object name 项目名称 -> id (com.duapp.dage)->crate project

https://build.phonegap.com/

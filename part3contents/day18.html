VUE: 是个MVVM框架
    MVC:衍生出很多变体  MVP MVVM MV*
    mv  vm~C
    MVVM M<->VM<->V
基本使用：
    new vue实例，传一堆配置，建议：只做一个根实例(vm)
    vm=new Vue({配置})  返回的实例
        配置:
            选项:值
            el: 要控制器的元素, 给个选择器
            data: 放置数据的地方
            methods:{
                方法:function(){
                    this == vm
                    this.数据名
                    this.方法名
                }
            }
    M:
        data 选项  是个对象
    V:  数据绑定|数据展示    1. 插值表达式  2. 属性绑定
        1. 插值表达式
        {{数据|数据名|属性|变量}}    模板标记 mustche 插值表达式
        2.绑定到属性：  指令="数据"  v-bind:html属性="数据"
        v-text="数据名"        vue特有的属性(指令)
        v-html="strong"        非转义输出
        
        v-bind:html属性="数据"    普通的html属性绑定数据
            :html属性="数据"  简写
        按需展示：
            v-for="(val,index) in 数据"        val值  index索引    变量数组、对象
            :key="item.id"  指定key 是数据id 修改VDom的key值
事件：事件名不带on
    v-on:事件名="方法"
       @事件名="方法"    简写
       @事件名="方法(参数)"
       @事件名="方法($event,参数)"        methods:{方法:function(ev,参数){ev/event}}    
               ev == event == 事件对象
               $event 传入的事件对象 可不传
模板表达式|插值表达式
计算属性:  是个函数|对象(get/set)  data数据变了，函数(get)就会调用（重新计算)
    选项里面:el / methods/ data
    computed:{
        计算属性:函数(){return 返回值}        使用:    {{计算属性}}
    }
    与method的区别:    方法会每次调用，计算属性不会(只有在与之相关的数据发生变化时，才调用)
    使用场景：查询

    公众平台生成公众号(订阅号、服务号、企业号)->不够用->用户自行开发(UI,数据库，空间都是自个的)->通过微信公众号|微信浏览器接入->使用jssdk调取设备信息

    为什么用：
        被关注,推送软文（文章）
        利用微信推广个人、或者企业
        使用微信服务
    基本服务: 摇一摇周边,投票管理,微信小店，微信支付..   
    模板选择，无需开发

jssdk:
    引入jssdk使用，微信服务、调取设备
    1) 申请空间+申请域名xx.duapp.com
        a) 注册百度账号,可以用百度云、百度、网盘
        b) 登录
        c) 部署空间->应用引擎BAE->添加部署->
    2) 测试空间&&域名
        svn仓库 绑定 本地目录  checkout commit
        删除密码文件：
            C:\Users\user\AppData\Roaming\Subversion\auth
    3) 服务器端配置(空间 DUAPP)    
        微信官网下载 wx_sample.php commit到服务器
        确认 是微信的后台
    4) 微信端配置
        微信公众号管理->公众号设置->功能设置->js接口安全域名->设置xx.duapp.com,不加http的        目的：我们的域名加入微信白名单
        下载一个 MP_veri...... .txt  不要打打开 commit 到 服务器
        微信公众号管理->开发->基本配置->修改配置
            url: http://xx.duapp.com/wx_sample.php
            token: 命名  == wx_sample.php 的token
    5) 下载开发示例代码
        开发者工具->开发文档->微信网页开发->js-sdk说明文档->拉到最下面->下在->copy php 到svn目录
        sample.php 就是开发实例文件  名字可以改
        修改sample.php 开发秘钥 == 微信秘钥(微信公共平台->开发-基本配置)    
        注意： 订阅号，不可用2017-9-29，用测试号，取消白名单，
    6) 调用js-sdk接口
        读文档: 开发者工具->开发文档->微信网页开发->js-sdk说明文档
https://mp.weixin.qq.com/wiki?t=resource/res_main&id=mp1421141115
        注意：订阅号，部分API可用   其他的要用，花钱（服务号，企业号）做完大家都可以看
              测试号, 几乎所有api可用    只有自个微信可以看
        使用API :
            wx.config 全局配置
                debug: true, // 调试模式
                jsApiList: [
                  // 所有要调用的 API 都要加到这个列表中
                  'chooseImage','scanQRCode','onMenuShareTimeline'
                ]
            wx.ready 处理js-sdk接口 成功
            wx.error 处理js-sdk接口 失败 会调用
        wx.
    申请测试号：

            接口配置信息:    
                URL:    http://uncle9.duapp.com/wx_sample.php
            测试号二维码: 微信一定要关注
    部署：
        公众号：自定义菜单->跳转网页->填入xx.duapp.com/php/sample.php
